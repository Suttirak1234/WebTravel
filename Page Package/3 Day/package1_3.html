แพ็คเกจที่ 1 : ตะลุยธรรมชาติและประวัติศาสตร์ (สระแก้ว-จันทบุรี)

  • • เวลา : 07:00 - 21:00 น.
  • • หมายเหตุ: การเดินทางระหว่างจังหวัด อาจใช้เวลาแตกต่างกันไป ขึ้นอยู่กับเส้นทางและสภาพการจราจร ควรเผื่อเวลาเดินทางไว้ด้วย ควรตรวจสอบข้อมูลสถานที่ท่องเที่ยวและเวลาเปิด-ปิด ก่อนการเดินทาง เพื่อความสะดวกและรวดเร็ว และควรเตรียมตัวให้พร้อมสำหรับการเดินทาง เช่น เสื้อผ้า รองเท้า อุปกรณ์ป้องกันแสงแดด และอื่นๆ ตามความเหมาะสม

กิจกรรม :

วันที่ 1: สระแก้ว (เน้นธรรมชาติ)

  • • 07:00 - 08:00 น. : เดินทางจากจุดนัดพบไปยังจังหวัดสระแก้ว (ใช้เวลาเดินทางประมาณ 2-3 ชั่วโมง ขึ้นอยู่กับจุดเริ่มต้น)
  • • 08:00 - 12:00 น. : เยี่ยมชมน้ำตกเขาชะเมา สัมผัสธรรมชาติอันอุดมสมบูรณ์ เดินป่าศึกษาธรรมชาติ ถ่ายรูป พักผ่อน
  • • 12:00 - 13:00 น. : รับประทานอาหารกลางวัน ณ ร้านอาหารใกล้เคียงน้ำตกเขาชะเมา
  • • 13:00 - 16:00 น. : เดินทางไปยังอุทยานแห่งชาติตาพระยา ศึกษาเรียนรู้ระบบนิเวศป่า ชมทิวทัศน์
  • • 16:00 - 17:00 น. : เช็คอินเข้าที่พักในอำเภอเมืองสระแก้ว
  • • 18:00 - 19:00 น. : รับประทานอาหารเย็นที่ร้านอาหารในตัวเมืองสระแก้ว
  • • 19:00 - 21:00 น. : พักผ่อนตามอัธยาศัย

วันที่ 2: จันทบุรี (เน้นประวัติศาสตร์และวัฒนธรรม)

  • • 07:00 - 08:00 น. : รับประทานอาหารเช้าที่โรงแรม
  • • 08:00 - 10:00 น. : เดินทางจากสระแก้วไปยังจังหวัดจันทบุรี (ใช้เวลาเดินทางประมาณ 2-3 ชั่วโมง)
  • • 10:00 - 12:00 น. : เยี่ยมชมวัดพระศรีมหาธาตุวรมหาวิหาร ชมความงดงามของสถาปัตยกรรม ศึกษาประวัติศาสตร์
  • • 12:00 - 13:00 น. : รับประทานอาหารกลางวันที่ร้านอาหารในตัวเมืองจันทบุรี
  • • 13:00 - 15:00 น. : เยี่ยมชมพิพิธภัณฑสถานแห่งชาติจันทบุรี เรียนรู้ประวัติศาสตร์และวัฒนธรรมท้องถิ่น
  • • 15:00 - 17:00 น. : เดินเที่ยวชมเมืองจันทบุรี ช้อปปิ้งสินค้าท้องถิ่น
  • • 17:00 - 18:00 น. : เช็คอินเข้าที่พักในจันทบุรี
  • • 18:00 - 19:00 น. : รับประทานอาหารเย็นที่ร้านอาหารทะเลสดๆในจันทบุรี
  • • 19:00 - 21:00 น. : พักผ่อนตามอัธยาศัย

วันที่ 3: เดินทางกลับ

  • • 07:00 - 08:00 น. : รับประทานอาหารเช้าที่โรงแรม
  • • 08:00 - 09:00 น. : เดินทางกลับ
