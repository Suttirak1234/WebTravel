แพ็คเกจที่ 4 : ธรรมชาติและแหล่งเรียนรู้ (ระยอง-จันทบุรี)

  • • เวลา : 07:00 - 21:00 น.
  • • หมายเหตุ: การเดินทางระหว่างจังหวัด อาจใช้เวลาแตกต่างกันไป ขึ้นอยู่กับเส้นทางและสภาพการจราจร ควรเผื่อเวลาเดินทางไว้ด้วย ควรตรวจสอบข้อมูลสถานที่ท่องเที่ยวและเวลาเปิด-ปิด ก่อนการเดินทาง เพื่อความสะดวกและรวดเร็ว และควรเตรียมตัวให้พร้อมสำหรับการเดินทาง เช่น เสื้อผ้า รองเท้า อุปกรณ์ป้องกันแสงแดด และอื่นๆ ตามความเหมาะสม

กิจกรรม :

วันที่ 1: ระยอง (เน้นธรรมชาติ)

  • • 07:00 - 08:00 น. : เดินทางจากจุดนัดพบไปยังจังหวัดระยอง
  • • 08:00 - 10:00 น. : เยี่ยมชมอุทยานแห่งชาติเขาแหลมหญ้า-หมู่เกาะเสม็ด เดินศึกษาธรรมชาติ ถ่ายรูปทิวทัศน์
  • • 10:00 - 12:00 น. : เที่ยวชมสวนผลไม้ในระยอง ชิมผลไม้ตามฤดูกาล
  • • 12:00 - 13:00 น. : รับประทานอาหารกลางวัน ณ ร้านอาหารในระยอง
  • • 13:00 - 15:00 น. : เยี่ยมชมไร่องุ่นซิลเวอร์เลค ถ่ายรูปและลิ้มรสไวน์
  • • 15:00 - 17:00 น. : เดินทางไปยังที่พักในระยอง เช็คอิน
  • • 18:00 - 19:00 น. : รับประทานอาหารเย็น ณ ร้านอาหารริมทะเล
  • • 19:00 - 21:00 น. : พักผ่อนตามอัธยาศัย

วันที่ 2: จันทบุรี (เน้นวัฒนธรรม)

  • • 07:00 - 08:00 น. : รับประทานอาหารเช้าที่โรงแรม
  • • 08:00 - 10:00 น. : เดินทางไปยังเมืองจันทบุรี เยี่ยมชมโบสถ์วัดแม่พระปฏิสนธินิรมล
  • • 10:00 - 12:00 น. : เดินเล่นและช้อปปิ้งสินค้าพื้นเมืองที่ตลาดพลู
  • • 12:00 - 13:00 น. : รับประทานอาหารกลางวัน ณ ร้านอาหารในจันทบุรี
  • • 13:00 - 15:00 น. : เยี่ยมชมอุทยานแห่งชาติน้ำตกพลิ้ว สัมผัสธรรมชาติ
  • • 15:00 - 17:00 น. : เดินทางกลับที่พักในจันทบุรี
  • • 18:00 - 19:00 น. : รับประทานอาหารเย็นที่ร้านอาหารทะเล
  • • 19:00 - 21:00 น. : พักผ่อน

วันที่ 3: เดินทางกลับ

  • • 07:00 - 08:00 น. : รับประทานอาหารเช้าที่โรงแรม
  • • 08:00 - 09:00 น. : เดินทางกลับ
